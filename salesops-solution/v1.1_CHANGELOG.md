# v1.1 Changelog ‚Äî rollback ledger

Every edit applied as part of the v1.1 build cycle (BUILD_QUEUE.md tasks).
Each task block lists every file touched and every code block added with
line-range markers. Inline markers in source are paired comments:
`# === v1.1 TASK-N START ===` and `# === v1.1 TASK-N END ===`.

## Rollback recipe (per task)

```bash
# Find every block introduced by a task
grep -rn "v1.1 TASK-N" backend/ frontend/

# To revert TASK-N:
#  1. Remove every block enclosed by the matched START/END markers.
#  2. Drop the DB columns listed under "DB migrations" below for that task.
#  3. Remove the seed data with namespace key prefixes listed under "KB seeds".
#  4. Remove any new files listed under "Files added" for that task.
```
